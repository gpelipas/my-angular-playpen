# Angular Project: Basic Bookmarks Collection  

Basic Angular Application that implements CRUD and REST API consumption

Building and running this project have few constraints and its workaround:

<br/>

Constraint:
 - Node (NPM) version is already old (version 17)
 - Angular CLI version is also old (version 18)

<br/>
 
Workaround:
- Install NVM (Node Version Manager) to be able to switch to multiple versions of Node (along its bundled NPM)
- Remove global Angular CLI and install specific version per project (in this case version 18)
